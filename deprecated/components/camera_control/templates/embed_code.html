🔧 Embedding Options

Option 1: iFrame Embed (Recommended)

The easiest way to embed the video stream. Just copy and paste this code into your HTML:

<iframe 
  src="http://{{ host }}/embed" 
  width="640" 
  height="480" 
  frameborder="0" 
  allowfullscreen>
</iframe>

Option 2: Direct Image Embed

Embed the video feed as a direct image (works in most browsers):

<img 
  src="http://{{ host }}/video_feed" 
  alt="Live Video Stream" 
  style="max-width: 100%; height: auto;">

Option 3: Responsive iFrame

A responsive embed that maintains aspect ratio (16:9):

<div style="position: relative; padding-bottom: 56.25%; height: 0; overflow: hidden;">
  <iframe 
    src="http://{{ host }}/embed" 
    style="position: absolute; top: 0; left: 0; width: 100%; height: 100%;" 
    frameborder="0" 
    allowfullscreen>
  </iframe>
</div>

Option 4: JavaScript/Fetch Integration

For advanced use cases where you need to process frames in JavaScript:

<img id="video-stream" alt="Live Video Stream">

<script>
  // Simple image-based approach
  document.getElementById('video-stream').src = 'http://{{ host }}/video_feed';
</script>

⚙️ Customization

You can customize the embed by adjusting these parameters:

  • width and height: Set the dimensions of the video player
  • frameborder: Set to "0" for no border, or "1" for a border
  • allowfullscreen: Allows the video to be viewed in fullscreen mode

📡 Direct Endpoints

Use these endpoints directly in your application:

  • Video Feed: http://{{ host }}/video_feed
  • Embed Player: http://{{ host }}/embed
  • Main Page: http://{{ host }}/
Note: Make sure to replace {{ host }} with your actual server address when deploying. If you're accessing from outside the Docker container, you may need to use your host machine's IP address instead of localhost.
